Disclaimer - ZapTurbo

Este projeto utiliza a biblioteca não oficial whatsapp-web.js, que automatiza interações com o WhatsApp Web via Puppeteer.

Avisos importantes:

  • Este projeto não é afiliado ao WhatsApp Inc.
  • O funcionamento depende da interface pública do WhatsApp Web, que pode mudar a qualquer momento sem aviso prévio.
  • Existe risco de bloqueio temporário ou permanente do número de WhatsApp utilizado.

Como reduzir o risco:

  • Use números dedicados, nunca seu WhatsApp pessoal
  • Evite spam ou mensagens em grande volume
  • Simule comportamento humano (pausas entre mensagens)

Para aplicações comerciais, considere utilizar a API oficial do WhatsApp Business.
